## Rank Math SEO API (confirmed working)

- Endpoint: `POST /wp-json/rankmath/v1/updateMeta`
- Payload: `{"objectID": POST_ID, "objectType": "post", "meta": {"rank_math_title": "...", "rank_math_description": "...", "rank_math_focus_keyword": "..."}}`
- Success response: `{"slug":true,"schemas":[]}`
- Available endpoints: `updateMeta`, `updateSchemas`, `updateMetaBulk`, `updateRedirection`, `getFeaturedImageId`
- Other endpoints return 404: `getResults`, `saveModule`, `updateSeoScore`, `updateSettings`, etc. (most are admin-only)
